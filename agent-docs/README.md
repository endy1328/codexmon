# 에이전트 하네스 문서

이 디렉터리는 `codexmon`의 에이전트 하네스 운영 문서를 보관한다.

문서 분류:
- `planning/`: 계획 로그와 작업 인벤토리
- `status/`: 현재 상태 기록
- `validation/`: 검증 기록
- `decisions/`: 판단 및 의사결정 이력

정본 제품/설계 문서는 이 디렉터리가 아니라 루트 `docs/` 아래에 둔다.

상태 모니터:
- `status/progress-monitor.html`: 작업 진행 모니터 HTML
- `status/progress.json`: 모니터가 읽는 구조화된 진행 스냅샷
